---
title: Closest Step Selector
published: 2026-03-05T22:23:01+00:00
---

The closest step selector selects the row of interest assuming a step-wise paradigm (i.e. selects the largest independent value smaller than the target value).

The closest step selector takes a target independent value as an argument and a dataset matrix as input.

The closest step selector assumes a step-wise paradigm.
