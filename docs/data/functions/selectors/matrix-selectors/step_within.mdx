title: step_within Selector

---

The step_within selector reduces the input matrix by removing all rows not within the given range of independent values. In other words, the output (rows of interest) will only have independent values within the given range.

The step_within selector takes the lower and upper bound of the range as arguments and takes a dataset matrix as input.

The step_within selector assumes a step-wise paradigm.
